Programação Web I – Trabalho de HTML
Descrição

Este repositório contém a atividade desenvolvida para a disciplina de Programação Web I.

O trabalho consistiu em reproduzir a estrutura e a formatação de um artigo acadêmico em HTML utilizando apenas as tags apresentadas em sala de aula (slides 1, 2 e 3).

O artigo utilizado foi:

"Assistente Virtual Inteligente para Instituições Educacionais: Aplicação da Técnica RAG no IFBot"

Objetivos da Atividade

Praticar a criação de páginas HTML.
Utilizar corretamente as principais tags estruturais.
Reproduzir a organização de um artigo acadêmico.
Implementar um índice de navegação utilizando âncoras.
Publicar o projeto utilizando o GitHub.

Recursos Utilizados

Estrutura completa do artigo em HTML.

Índice de navegação com links internos (âncoras).
